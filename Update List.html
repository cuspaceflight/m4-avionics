Component upgrades

Several components in m4-core-avionics are deprecated, and therefore no longer sold by Farnell.
This is a list of suggested replacements. The various schematics should be updated shortly.

1. Each schematic in m4-core-avionics had a "Bill of Materials" generated.
2. These files were then searched to produce the list of Farnell order numbers used.
3. The associated Farnell web pages were then visited.
4. The following set of criteria were applied to determine if a component should be upgraded:
  • Any component out of stock/no longer stocked/..... was upgraded.
  • Any component which was "on sale until stocks were exhausted" was upgraded, on the grounds that, by the time Martlet 4 flies, they will be out of stock.
  • Conversely, any component on back-order was ignored, on the grounds that the back-order will have arrived by the time the rocket is being assembled, they should be back in stock.
5. Those components which had to be upgraded had upgrades chosen on a case-by-case basis. Frequently Farnell would suggest a replacement, or otherwise a replacement which matched the necessary requirements could be found using Farnell custom search.
No components were harmed in the making of this report.
